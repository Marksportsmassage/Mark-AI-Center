## Basic Principles From Source

The source distinguishes common scoring issues:

- 5 分到 4 分: 施壓部位不同。
- 3 分: 要給阻力 for selected motions.
- 3 分: 要去重力 for selected motions.
- 2 分: 要給阻力 for ankle plantarflexion.
- 2 分到 1 分: 去重力姿勢不同.
- 2 分: 部分活動度.
- 分開測試: wrist flexion/extension, knee flexion, neck flexion.

## Wrist And Hand

### Wrist Flexion

- 作動肌肉: 橈側屈腕肌, 尺側屈腕肌.
- Nerve in source: 正中神經 for 橈側屈腕肌; 尺神經 for 尺側屈腕肌.
- 5-3 分: short sitting, elbow flexed, forearm supinated, supported on table, fingers relaxed, perform wrist flexion.
- Resistance: palm side, downward resistance.
- 2 分: forearm neutral, table supports wrist.
- 1-0 分: forearm supinated, table supports wrist.
- Note: must test separately.

### Wrist Extension

- 作動肌肉: 橈側伸腕長/短肌, 尺側伸腕肌.
- Nerve in source: radial nerve.
- 5-3 分: short sitting, elbow flexed, forearm pronated, supported on table, fingers relaxed, perform wrist extension.
- Resistance: dorsum of hand, downward resistance.
- 2 分: forearm neutral, table supports wrist.
- 1-0 分: forearm pronated, table supports wrist.
- Note: must test separately.

## Hip Flexion

### Simple Hip Flexion

- 作動肌肉: 腰大肌 + 髂肌.
- Nerve in source: lumbar nerve + femoral nerve.
- 5-3 分: short sitting, pelvis neutral, lift thigh from table.
- Resistance: middle thigh, downward resistance.
- 2 分: side lying with therapist supporting knee.
- 1-0 分: supine with therapist supporting knee.
- Note: two gravity-eliminated postures.

### Hip Flexion + Abduction + External Rotation + Knee Flexion

- 作動肌肉: 縫匠肌.
- Nerve in source: femoral nerve.
- 5-3 分: short sitting, slide foot up along opposite tibia.
- Resistance: lateral knee and medial lower leg, downward/outward resistance.
- 2-0 分: supine with therapist supporting knee.
