## Key Points

- 止痛機轉：低頻偏 opioid / spinal mechanism，高頻偏 gate control。
- Gate control：Aβ 大纖維促進膠狀質細胞抑制 T cell；C fiber 傳痛覺。
- 高頻止痛快但較短暫；低頻止痛較慢但可維持較久。
