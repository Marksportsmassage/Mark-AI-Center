# Final Review Plan

Goal: produce subject-specific review files that are accurate, source-linked, and useful for final exam cramming.

## Output Structure

Each subject should eventually include:

- Scope
- High-yield review
- Question bank
- Answer key
- Must-memorize tables
- Final 30-minute review

## Extraction Order

1. Surgery question bank: extract all questions, options, answers, explanations, and page references.
2. ROM question bank: classify by body region and Norkin principle.
3. MMT answer PDFs: build muscle-action-innervation tables.
4. Physical Modality lecture PDFs: extract definitions, mechanisms, parameters, indications, contraindications, and lecture-highlighted points.
5. Operation Therapy PDFs: extract readable operation flows, contraindications, precautions, cues, and mistakes.

## Quality Rules

- Original question requires source file and page number.
- Lecture point can be marked as "lecture high-yield", not "original question".
- Missing answer should be marked `待補`, not guessed.
- Scanned unreadable pages should be listed for Mark review.
