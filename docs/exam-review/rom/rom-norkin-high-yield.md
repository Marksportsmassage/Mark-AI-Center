## 已可確認重點

- MCP / PIP / DIP 屈曲題目需依 Norkin 量測原則判斷。
- DIP 屈曲量測時，PIP 應保持屈曲 70-90 度。
- 指間關節量測時，要控制近端關節位置，避免肌腱張力干擾。
- 拇指 CMC 屈伸、外展與量角器支點/近端臂/遠端臂是高頻混淆點。
- ROM 講義目前是掃描型 PDF，需 Mark 提供可 OCR 版本或人工確認表格。
