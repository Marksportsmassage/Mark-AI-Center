## Scoring Standard

Run-station time: 3 minutes.

Each ROM item is scored by whether Mark can:

1. 將受測者擺在測試擺位。
2. 在適當固定下做測試動作。
3. 說出做測試動作時的終點感覺。
4. 說明量角器支點應置放的骨骼標記，說明如何找出，確實找出並用筆標出。
5. 說明量角器固定臂如何置放；若有參考點，說明參考點、找出並標出。
6. 說明量角器移動臂如何置放；若有參考點，說明參考點、找出並標出。
7. 測量角度時正確放置支點、固定臂、移動臂。
8. 檢測者站在合適位置，眼睛和量角器同一高度。
9. 確認病人關節活動度完整且沒有代償。
10. 讀出正確刻度。
